# Ramen: Robust Test-Time Adaptation of Vision-Language Models with Active Sample Selection

## 프로젝트 소개

본 저장소는 **"Ramen: Robust Test-Time Adaptation of Vision-Language Models with Active Sample Selection"** 논문을 읽고 구현하며 내용을 학습하기 위한 프로젝트입니다.

Ramen은 Vision-Language Model(VLM)의 Test-Time Adaptation(TTA) 성능을 향상시키기 위해 Active Sample Selection 기법을 적용한 방법을 제안합니다. 본 프로젝트에서는 논문의 핵심 아이디어와 알고리즘을 이해하고 재현하는 것을 목표로 합니다.
